# 資料科學與 R 語言

這裡是東吳大學 110 學年度第二學期「資料科學與 R 語言」的課程主頁。

## 評分標準

繳交資料分析或其他研究問題之（可正確編譯的） Rmd 檔案與輔助資料檔，以壓縮檔方式於 6/24 （五）中午十二點前寄至授課老師電子信箱。
